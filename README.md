# Billing_information
Develop a basic web-based UI using express.js and HTML,CSS,Javascript . the UI consist of single page with button ,"Get Billing information".

## PROJECT SETUP

```bash
  npm init -y
```

```bash
  npm i express
```
```bash
  npm i aws-sdk
```
```bash
  npm i body-parser
```
```bash
  npm i cors
```

## RUN
## Run Server


```bash
  node app.js
```
## After clone your repository
```bash
  ng serve
```
